# روتين التحليلات والمتابعة الشهرية

نفّذ في **آخر أسبوع عمل** من الشهر أو أول يوم من الشهر التالي.

## 1. إقفال تكلفة الإنتاج

| الخطوة | المسار | ملاحظات |
|--------|--------|---------|
| مراجعة التقارير | `/reports` | اكتمال الشهر، تصحيح أخطاء |
| صحة البيانات | `/costs/health` | صفر مشاكل حرجة قبل الإقفال |
| مراكز التكلفة | `/cost-centers` | قيم الشهر محدثة |
| إقفال الشهر | `/monthly-costs` | إقفال الفترة `YYYY-MM` |
| تحليل الانحراف | `/monthly-costs` | قسم التحليل الذكي + مقارنة بالشهر السابق |

**علامة نجاح:** متوسط تكلفة الوحدة على لوحة Admin/Factory يعكس الشهر المغلق (وليس 0.00 فقط).

## 2. تحليلات الإنتاج

| التحليل | المسار | متى |
|---------|--------|-----|
| ملخص المنتجات | `/` Admin → جدول المنتجات | بعد الإقفال |
| مسارات الإنتاج | `/production/routing/analytics` | منتجات لها routing |
| تخطيط احتياجات | `/manufacturing/planning-run` | قبل موسم/ذروة قادمة |

## 3. مخزون

| الخطوة | المسار |
|--------|--------|
| جرد دوري (إن مطلوب) | `/inventory/counts` |
| مراجعة قيمة المخزون | `/inventory` |
| تصدير أرصدة | `/inventory/balances` → Excel |

**غير متوفر في النظام:** تقرير ABC، دوران مخزون، تقادم — استخدم Excel من التصدير.

## 4. أصول وإهلاك

| الخطوة | المسار |
|--------|--------|
| تشغيل إهلاك الشهر | `/costs/assets` |
| تقرير الإهلاك | `/costs/depreciation-report` |

## 5. أرشفة ومتابعة

- [ ] تصدير تكلفة شهرية Excel من `/monthly-costs`.
- [ ] حفظ لقطة من `/costs/health` إن وُجدت مشاكل مُعالجة.
- [ ] مراجعة خطط الشهر القادم في `/production-plans`.

## جدول زمني مقترح

| اليوم | النشاط |
|-------|--------|
| 25–28 الشهر | صحة تكاليف + مراجعة تقارير |
| 29–30 الشهر | إقفال + انحراف |
| 1–3 الشهر الجديد | تخطيط احتياجات + خطط إنتاج |

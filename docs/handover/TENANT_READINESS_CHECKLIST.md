## علامات نجاح التشغيل الكامل

- بعد تقرير إنتاج (تام/تعبئة): تظهر حركة أو طلب تحويل في `/inventory/transactions` أو `/inventory/transfer-approvals`.
- لوحة المخزون: قيمة تقديرية &gt; 0 (إن وُجدت أرصدة وتكاليف مواد).
- لوحة مدير النظام: متوسط تكلفة وحدة &gt; 0 بعد إقفال شهر أو بيانات كافية.
- صحة بيانات التكاليف: لا مشاكل **حرجة** في `/costs/health`.

## ما لا يغطيه النظام حالياً (خارج التشغيل)

- شراء مواد (PO) وتنفيذ MRP تلقائي عند إنشاء الخطة.
- مركز عمليات موحّد (inbox) لكل التنبيهات.
- تحليلات مخزون متقدمة (ABC، دوران، تقادم).
- تذكيرات بريد/push لتحويلات معلقة أو خطط متأخرة.
